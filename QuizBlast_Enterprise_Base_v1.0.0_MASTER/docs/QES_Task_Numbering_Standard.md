# QES — Task Numbering Standard

## Principle
Disiplin hızdan daha değerlidir.

## Format
`QB-<SPRINT>-T<NNN>`

Example: `QB-233-T001`

- `QB`: QuizBlast
- `233`: Sprint 2.3.3
- `T001`: Task 001
